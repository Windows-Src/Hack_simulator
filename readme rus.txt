Для того чтобы это работала надо чють исправить файл cmd и извлечь архив
для этого на нажать ПКМ "Изменить" и вписать cd C:\Users\Ваш пользователь\Desktop\Hack вместо Ваш пользователь впешите своё имя пользователя
если серовно не работает то зайдите в проводник и нажмите Локальный диск(С:) далее Пользователи (Users) и посмотрите ваше имя пользователя.
